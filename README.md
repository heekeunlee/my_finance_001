# My Finance

A premium, intuitive finance tracker web application inspired by Toss and Apple design.
Track your income, expenses, and savings with ease.

## Features
- **Dashboard**: View your total balance and financial breakdown at a glance.
- **Transaction Tracking**: Add income, fixed/variable expenses, and savings.
- **Savings Advisor**: Smart suggestions to improve your financial health based on your data.
- **Responsive Design**: Looks great on both mobile and desktop.
- **Data Persistence**: Data is saved locally in your browser.

## Tech Stack
- React
- Vite
- Vanilla CSS (Design Tokens)

## Deployment
This project is deployed to GitHub Pages.

## Usage
1. Open the app.
2. Add your income and expenses.
3. Check the dashboard for insights.
